le jeu du covid 19

règles


DÉPLACEMENT

Pour se déplacer, il faut utiliser les touches w,a,s et d ou les flèches.Votre personnage est le cercle gris avec des points rouges.


LE BUT DU JEU

Le but du jeu est de contaminer tous les cercles verts. En tout, il y en a 20. Pour cela, il faudra vous déplacer à proximité de ceux-ci. Une fois contaminés, les cercles verts deviendront rouges.


LES SEGMENTS PLUS

Il y a des frontières, ce sont les lignes noires, vous ne pouvez pas les dépasser, mais les cercles contaminés ou non le peuvent. Mais quand ils sont trop loin de la frontière, ils réapparaissent quelque part sur la carte.

Il y a une façon de lancer des boules. Pour cela, il faut bouger dans la direction dans laquelle vous voulez envoyer votre balle et appuyer sur "e". Quand un cercle contaminé touche un cercle non contaminé, les deux cercles deviennent contaminés.


LES DOCTEURS

Le docteur apparaît au niveau 2. Les docteurs sont représentés par des cercles blancs, ils peuvent soigner les cercles contaminés. Les docteurs ne peuvent en soigner que 5.


Les docs

Le doc apparaît au niveau 4. Les docs ressemblent aux docteurs sauf qu'ils ne bougent pas et sont plus gros.Ils peuvent soigner 20 personnes avant de disparaître.


Les docs bleus

Le doc bleu apparaît au niveau 7. Les docs bleus ont un point bleu au centre. Les docs bleus ressemblent aux docs sauf qu'ils bougent.


Les docs rouges

Le doc rouges apparaît au niveau 9. Les docs rouges ont un point rouge au centre. Les docs rouges ressemblent aux docs bleus sauf qu'ils grossissent quand ils décontaminent.


Les docs verts

Le doc vert apparaît au niveau 11. Les docs verts ont un point vert au centre. Les docs verts ressemblent aux docs rouges sauf qu'ils sont immortelles.


Les docteurs améliorés

Le docteur amélioré apparaît au niveau 5. Les docteurs améliorés ressemblent aux docteurs sauf qu'ils ont un point bleu au centre. Quand ils décontaminent quelqu'un ils disparaissent et apparaissent 20 secondes plus tard près d'un doc de n'importe quelle couleur.
